# 🚀 خطوات نشر HRFA.MA - دليل سريع

## ✅ الخطوة 1: نشر Backend على Railway (5 دقائق)

**الرابط:** https://railway.app

1. سجل دخول بـ GitHub
2. اضغط "New Project" → "Deploy from GitHub"
3. اختر: `mohaaitbensaid-coder/hrfFa`
4. أضف المتغيرات التالية في **Variables**:

```env
PORT=5000
NODE_ENV=production
MONGODB_URI=سيتم إضافته من MongoDB Atlas
JWT_SECRET=hrfa-secure-2026-production-key
JWT_EXPIRE=7d
CLIENT_URL=سيتم تحديثه بعد Vercel
```

5. احفظ رابط Railway (مثل: `https://web-production-xxxx.up.railway.app`)

---

## 📊 الخطوة 2: إعداد MongoDB Atlas (5 دقائق)

**الرابط:** https://mongodb.com/cloud/atlas

1. أنشئ حساب مجاني
2. "Create Cluster" → اختر **Free (M0)**
3. "Database Access" → أنشئ مستخدم (احفظ Username & Password)
4. "Network Access" → أضف `0.0.0.0/0`
5. "Clusters" → "Connect" → "Connect your application"
6. انسخ Connection String:
   ```
   mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/hrfa-ma
   ```
7. ارجع لـ Railway → Variables → أضف في `MONGODB_URI`

---

## 🎨 الخطوة 3: نشر Frontend على Vercel (3 دقائق)

**الرابط:** https://vercel.com

1. سجل دخول بـ GitHub
2. "Add New Project" → اختر `mohaaitbensaid-coder/hrfFa`
3. في Configure Project:
   - **Framework:** Vite
   - **Root Directory:** اضغط "Edit" → اكتب `frontend`
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`

4. في **Environment Variables** أضف:
   ```
   VITE_API_URL=https://web-production-xxxx.up.railway.app/api
   ```
   (استخدم رابط Railway من الخطوة 1)

5. اضغط **Deploy** وانتظر

---

## 🔄 الخطوة 4: تحديث CORS (دقيقة واحدة)

1. ارجع لـ Railway Dashboard
2. Variables → حدث `CLIENT_URL` إلى رابط Vercel:
   ```
   CLIENT_URL=https://your-app.vercel.app
   ```
